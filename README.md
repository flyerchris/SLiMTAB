## Run
SLiMTab requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and devDependencies and start.

```sh
$ git clone https://github.com/tim37021/SLiMTAB
$ cd SLiMTAB
$ npm install
$ git clone https://github.com/ps2747/SLiMTAB-backend
$ npm start
```
